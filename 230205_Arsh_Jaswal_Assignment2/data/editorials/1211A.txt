1211A - Three ProblemsJust choose any three indices with different values and print them in order of increasing values. You can do it in a billion ways. One of the is to use map data structure (from elements to indices): while reading the data assign value $$$i$$$ for the key $$$a_i$$$ (use an ordered map, for example, TreeMap). If the size of the map less than $$$3$$$, print -1. Overwise print values (i.e. correspondent values) for the first three elements of the map.Here is the example of a solution in Kotlin:fun main() {    val n = readLine()!!.toInt()    val r = readLine()!!.split(" ").map { it.toInt() }        .withIndex()        .distinctBy { it.value }        .sortedBy { it.value }        .map { it.index + 1 }    if (r.size < 3) {        println("-1 -1 -1")    } else {        println("${r[0]} ${r[1]} ${r[2]}")    }}
