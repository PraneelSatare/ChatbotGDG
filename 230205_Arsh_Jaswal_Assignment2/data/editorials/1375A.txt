1375A - Изменение знаковNotice that $$$a_{i + 1} - a_i \ge 0$$$ is equivalent to $$$a_{i + 1} \ge a_i$$$. Similarly $$$a_{i + 1} - a_i \le 0$$$ is equivalent to $$$a_{i + 1} \le a_i$$$. Flip the signs in such a way that $$$a_i \ge 0$$$ for odd $$$i$$$, while $$$a_i \le 0$$$ for even $$$i$$$. Then  $$$a_{i + 1} \ge 0 \ge a_i$$$, and thus $$$a_{i + 1} \ge a_i$$$, for $$$i = 2, 4, \dots, n - 1$$$.  $$$a_{i + 1} \le 0 \le a_i$$$, and thus $$$a_{i + 1} \le a_i$$$, for $$$i = 1, 3, \dots, n - 2$$$.
