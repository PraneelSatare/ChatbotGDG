# Codeforces Scraper

This Python script scrapes problem details from Codeforces. It extracts basic information such as the problem title, time limit, memory limit, problem statement, sample test cases, and tags. The data is saved in a JSON format.

## Features

- Scrapes problem title, time limit, memory limit, and problem statement.
- Extracts sample input/output test cases.
- Gathers problem tags.
- Saves the scraped data as a JSON file for each problem.
